== Description ==

Ajaxified Contact Form widget can be used in your wordpress site to send contact form. Users will be able to contact you through this form dynamically. It's a simple Ajax Powered contact form

== Installation ==

1. Upload the plugin through `Plugins > Add New > Upload` interface or upload `ajaxified-contact-form` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. In the widgets section, select the Ajaxified Contact Form widget and add to one of your widget areas
4. Set the required settings and save your widget
